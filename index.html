Description

OrganizeIt is an app that let's the user organize weekly meetings, classes, and events. The app helps organizing the events to prevent having overlap in their schdule so that the user will not get stuck with two meetings at the same time. Also users will be notified when the event starts so they can get reminded for it. OrganizeIt can help you to not forget your important meetings and deadlines. This app is developed by Asem Shaath
